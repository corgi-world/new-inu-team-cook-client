# 인천대학교 캡스톤 디자인

## 개요

- 실시간으로 이슈 된 키워드의 상세한 정보를 한눈에 확인할 수 있는 서비스
- 구두 발표로 선정 및 장려상 수상
- 프론트엔드 파트의 모든 개발과 배포를 담당
